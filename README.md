md2fswiki.vim
=============

markdown to FreeStyleWiki.

call [md2fswiki](https://github.com/ywatase/md2fswiki) from vim

## Usage

```
:Md2fswiki
```

## Setting

```
" set md2fswiki path
" default: md2fswiki
let g:md2fswiki_path = 'md2fswiki'
```

## Install md2fswiki

requires

* perl
* cpanm

```
make
```
